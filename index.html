  • input: connected, undirected weighted graph G=(V,E)
  • output: A spanning tree T

Prm’s Algorithm

  1. select a “root”
  2. grow a tree from the root by selecting the least-weight edge with one vertex already in tree that building

correctness : use “cut”

  • DEF: A cut is any partition of vertices V into two nonempty sets, S and V-S.
  1. The two cells of set $C_1$ and $C_2$ are disjoint, $C_1\cap C_2=\emptyset$
  2. $C_1\cup C_2=V$
  • CUT Property:
  1. Let G=(V,E) be a connected, undirected, weighted graph.
  2. Suppose edges $X\subseteq E$ are part of an MST T of G
  3. pick any nonempty subset S of vertinces V, $S\subseteq V$, such that no edge in X crosses between S and V-S
  4. Let $e\in E$ be an edge of minimum weight across this CUT(S, V-S), then $X\cup{e}$ is part of some MST of G, i.e. $X\cup{e}\subseteq T’$, where T’ is an MST of G
  • proof:

    edges X are part of some MST T of G. e=(u,v), edge of min weight across cut

  1. CASE 1: if the new edge $e\in T$ also, then there is nothing to prove
  2. CASE 2: assume $e\notin T$, will construct a different MST T’ containing $X\cup{e}$
    add edge e to T: $T\cup{e}$
    since T is connected and spans G, T already contains a path from u to v, so adding e=（u,v）creates a cycle
    This cycle must have some other edge e’ across CUT(S, V-S)
    if we remove this edge e’, we get $T’ = T \cup {e}-{e’}$ show T’ is a tree
    T’ is connected, since T is connected and e’ is a cycle edge
    T’ has the same number of edges as T, so T’ is a tree.
    Moreover, T’ is an MST
    Because w(T’)=w(T)+w(e)-w(e’), both e and e’ cross the CUT(S,V-S), and e is epcifically the min weight edge of this type
    So, w(e)<=w(e’), w(T’)<=w(T) -> w(T’)=w(T)
    Thus, T’ is also an MST

Inverse Transform Sampling

wiki

  • Technique to select a random value from an arbitrary CDF (or PDF)
  • Very efficient when it works analytically
  • Often good numerically but not always

IVT Algorithm

  1. select U from v[0,1] i.e. uniformly
  2. find the value X such that
    F(x)=U where F is the CDF
    i.e. solve $X=F^{-1}(U)$

example

cosider PDF: $p(x)=e^{-x}$ where $0<=x<\infty$

  1. Find CDF:
    $F(x)=1-e^{-x}$
  2. Invert CDF:
    $F^{-1}(U)=-log(1-U)$
  3. select a U and evaluate $F^{-1}(U)$

Rejection Sampling

wiki

  • suppose you have a method of sampling from some PDF, call it g(x) and you want to draw a sample from some other PDF, f(x)
  • if for some constant M, $f(x)<=Mg(x)$

REJ Algorithm

  1. sample X from g(x)
  2. calculte $\alpha$, the probability of accepting X
    $$\alpha=\frac{f(x)}{Mg(x)} \,\, (note \,\, 0<=\alpha<=1)$$
  3. sample U from U(0,1]
  4. if $\alpha$>=U accept, else reject
  5. repeat

Propability

Sample space and Events

  • let S = { all possible ordings 1…7 }
    S:”Sample Space”
  • let A be a subset of S, A is called an “event”
  • “union”: A $\cup$ B all outcomes in either A, B, or both
  • “intersection”: A $\cap$ B in both A and B
  • A and B are “disjoint” or “exclusive” if $A\cap B=\varnothing $
  • $A^c$ is A’s complement

    Axioms

  • 0<=P(A)<=1
  • P(S)=1
  • for any mutully exclusive events $ A_1,A_2,…,A_n $:
    $$ P(\bigcup_{i=1}^n A_i) = \sum_{i=1}^n P(A_i) $$
  • generally:
    $$P(A\cup B)=P(A)+P(B)-P(A\cap B)$$

    Conditional Probability

    Probability of A given B
    $$ P(A|B) = \left.\frac{P(A\cup B)}{P(B)}\right. $$

Random Variables

  • In an experiment the unknown outcome is designeted as a random variable.

    Cumulative distribution function

  • For a random process, this outcome cannot be predicted, however we may know the probability of a perdict outcome. This is described by a cumulative distribution function (CDF)
    $$F(X)=P{ \mathbb{X} <= x }$$
    Where $\mathbb{X}$ is a given experiment and x denotes all possible outcomes.
    Laplace’s demon

Probability density function

  • The PDF is an equally common tool that serves the same purpose as the CDF.
  • The PDF is defined as the dervative of the distribution function
    $$ P(x)=\left.\frac{dP(x)}{dx}\right.$$
  • For a discrete sample space $P(x_i)$ is intepreted as $P(x_i)=P(\mathbb{X}=x_i)$
    Error function
    Mexican hat wavelet

PDF for continues random variable

$$P(x \in C)= \int_{C} P(x)dx$$
$$ P(a-\frac c2<=x<=a+\frac c2)=\int_{a-\frac c2}^{a+\frac c2} P(x)dx$$
$$ \int_{-\infty}^{\infty} P(x)dx=1$$
$$ P(x)>=0\,\,\, \forall x$$

Expectation

The expected value of a random variable E[x] is defined as

  • discrete: $E[x]=\sum_i x_iP(x_i)$ where P(x) is the discrete PDF of x
  • continuous: $E[x]=\int_{-\infty}^\infty x_iP(x_i)$

  1. Tree/forward Edge : d[u]<d[v]<f[v]<f[u]

Strongly Connected Components

  • DEF: two vertices u and v of a directed graph G are strongly connected if there is a path from u to v and a path from v to u
  • Partitions V in to disjoint sets of vertices called Strongly Connected Components(SCC)
  • A SCC is a maximal set of strongly connected vertices

    Clever (Tarjan): ideas

  1. consider transpose of G=(V,E): $G^T=(V,E^T)$ where $E^T={(v,u): (u,v) \in E}$
    observation: G and $G^T$ have the same SCCs
  2. Compdnent Graph
    $G^{SCC}=(V^{SCC},E^{SCC})$
    $V^{SCC}$: one vertex for each SCC in G
    $E^{SCC}$: has an edge if there is anedge between SCCs in G

proof:

let X and X’ be distinct SCCs in G
let $u,v \in X$, $u’,v’ \in X’$, and suppose there is a path from u to u’ in G, then there cannot be a path from v’ to v in G

How do we find a vertex in a sink SCC?

  • NOTE: The vertex with lowest finishing time might not be in a sink

Shortest Path

Weighted Directed Graphs

  • A weighted graph H is a directed together with a weight function
    $G=(V,E,w)$
  • shortest path from u to v is a path of minimum weight from u to v
  • shortest path weight from u to v:
    $\delta$(u,v)=min{ w(P) : P from u to v }

    When do shortest paths not exist?

    Negative edge weights
    Negative weight cycle:
    $\delta$(u,v) = $-\infty$
    if not reachable, $\delta$(u,v) = $+\infty$

Dijkstra’s Algorithm

  • input: ( G, w, S)
    • G=(V,E) Directed, ADJ
    • Weight function: w: E->R ( $\forall e \in E$ )( w(e)>=0 )
    • S in V, Source vertex
  • output:
    • d[v]: distance (weight of a shorest path from S to V)
      d[v] = $\delta$(S,V)
    • $\pi$[v]: parent of V on shorest paths tree

    suppose u has just been delected from Q, adn for all vertices w previously delected from Q, d[u]=$\delta$(s,u)

    Then x=$\pi$[y] $\in$ S, so by IH d[x]=$\delta$(s,x)
