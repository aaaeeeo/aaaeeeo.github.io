Sample space and Events

  • let S = { all possible ordings 1…7 }
    S:”Sample Space”
  • let A be a subset of S, A is called an “event”
  • “union”: A $\cup$ B all outcomes in either A, B, or both
  • “intersection”: A $\cap$ B in both A and B
  • A and B are “disjoint” or “exclusive” if $A\cap B=\varnothing $
  • $A^c$ is A’s complement

    Axioms

  • 0<=P(A)<=1
  • P(S)=1
  • for any mutully exclusive events $ A_1,A_2,…,A_n $:
    $$ P(\bigcup_{i=1}^n A_i) = \sum_{i=1}^n P(A_i) $$
  • generally:
    $$P(A\cup B)=P(A)+P(B)-P(A\cap B)$$

    Conditional Probability

    Probability of A given B
    $$ P(A|B) = \left.\frac{P(A\cup B)}{P(B)}\right. $$

Random Variables

  • In an experiment the unknown outcome is designeted as a random variable.

    Cumulative distribution function

  • For a random process, this outcome cannot be predicted, however we may know the probability of a perdict outcome. This is described by a cumulative distribution function (CDF)
    $$F(X)=P{ \mathbb{X} <= x }$$
    Where $\mathbb{X}$ is a given experiment and x denotes all possible outcomes.
    Laplace’s demon

Probability density function

  • The PDF is an equally common tool that serves the same purpose as the CDF.
  • The PDF is defined as the dervative of the distribution function
    $$ P(x)=\left.\frac{dP(x)}{dx}\right.$$
  • For a discrete sample space $P(x_i)$ is intepreted as $P(x_i)=P(\mathbb{X}=x_i)$
    Error function
    Mexican hat wavelet

PDF for continues random variable

$$P(x \in C)= \int_{C} P(x)dx$$
$$ P(a-\frac c2<=x<=a+\frac c2)=\int_{a-\frac c2}^{a+\frac c2} P(x)dx$$
$$ \int_{-\infty}^{\infty} P(x)dx=1$$
$$ P(x)>=0\,\,\, \forall x$$

Expectation

The expected value of a random variable E[x] is defined as

  • discrete: $E[x]=\sum_i x_iP(x_i)$ where P(x) is the discrete PDF of x
  • continuous: $E[x]=\int_{-\infty}^\infty x_iP(x_i)$
